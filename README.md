# Fitness Tracker aplikacija

Fitness Tracker je mobilna aplikacija izrađena za potrebe fakultetskog projekta.

## Tehnologije

- Mobile: React Native + Expo
- Backend: Node.js + Express
- ORM: Sequelize
- Baza: Supabase PostgreSQL

## Struktura projekta

- backend/ - backend server i API
- mobile/ - mobilna aplikacija
